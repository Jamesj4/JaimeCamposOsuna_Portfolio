⚙️ Modular Custom Base Plugin

This is a project I am currently developing with a colleague. We both handle various tasks, but primarily, I focus on the visual (using Slate Unreal Framework) and editor logic, while he works on the runtime logic. The goal is to develop multiple plugins based on our framework. We are already working on plugins like Dialogue Systems and Conditions, which are being built on top of this base. Following best programming practices, the foundation is modular and scalable. For example, the nodes are designed as objects, allowing users to inherit and customize them in Blueprint. Nearly everything in the plugin is customizable and adaptable to suit various needs.

⚙️ Plugin Features

This plugin is modular and scalable. We have developed several features within it, including a debug system that mimics Unreal's Behavior Tree system. This feature displays real-time, custom properties when hovering over objects. Additionally, we implemented a recursive node search system (which allows finding information within child nodes) designed to be as efficient as possible. Furthermore, we included a compiler that alerts the user of errors, enabling them to quickly navigate to the exact issue in the graph.
